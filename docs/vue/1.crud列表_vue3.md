# 解决页面不更新问题
问题：vite-vue除app.vue和main.js里面的代码变更有效，其他组件变更代码，页面没有更新，刷新也不行，必须重启项目
在网上看了一下解决方案是在配置里面开启热更新，配置了依旧无效

最后发现问题点在于引入组件大小写问题,我把Index.vue写成了index.vue
import CrudList from './views/crudList/index.vue'  正确
import CrudList from './views/crudList/Index.vue'  正确
